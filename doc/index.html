1 Plotting functions

To plot a function you simply type the function in the field at the bottom of the programs window and you can either press the Add button or the ENTER key.

To delete a function, select it in the function list and press the DELETE key.

1.1 Operators

This is a table of the available operators. The operator with the highest precedence will be evaluated first.

Precedence Operator Description Example
5 ^ Power of x^2
4 + - Unary plus and minus -x
3 * / Multiplication and division 10 * x
2 + - Plus and minus 10 + x

2 Tools

2.1 Move tool

To use the move tool, select it in the toolbar at the top of the window. Moving around is as simple as dragging around in the rendering area.

2.2 Zoom tool

The zoom tool has two modes: normal zoom or box zoom. To use normal zoom simply left click anywhere in the rendering area to zoom in and hold ALT key while clicking to zoom out.

To use box zoom press and hold your left mouse button somewhere in the rendering area, and move the cursor until you see a dashed box, finally release the mouse button.

2.3 Selection tool

First select a function from the function list, then press anywhere in the rendering area to select a point on the graph.
